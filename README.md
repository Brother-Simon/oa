## 简介

医院OA管理系统 是一个商业的，大型综合医疗集团OA业务平台系统 ，创立于2014年末，是为了Money而诞生的。医院OA管理系统 基于 ThinkPHP 3.2.2 框架 + Jquery EasyUI 1.4.1 框架开发
